## Online Examination System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/584c5a51e159400fa1b9c9480f108aff/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): students, teachers, administrators.

**Student Features:**
1. Login
2. Pending Exam List
3. My Exam Papers
4. My Practice
5. My Scores
6. Start Exam
7. Change Password
8. Leave Me a Message
9. Secure Logout

**Teacher Features:**
1. Login
2. Exam Query
3. Add Exam
4. Question Bank Management, Add Question Bank
5. Student Grade Query
6. Grade Segment Statistics
7. Student Management
8. Add Student
9. Secure Logout


**Administrator Features:**
1. Login
2. Exam Query
3. Add Exam
4. Question Bank Management, Add Question Bank
5. Student Grade Query
6. Grade Segment Statistics
7. Student Management
8. Add Student
9. Teacher Management
10. Add Teacher
11. Secure Logout
